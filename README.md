# Design Patterns examples

Various implementations of different design patterns.

## Introduction

This repository is my attempt to make sense of different design patterns, including tangible examples.
Every design pattern and code examples will be located into its own subfolder.

## Patterns

### Creational

- Abstract Factory
- Builder
- Factory Method
- Prototype
- [Singleton](singleton)

### Structural

- Adapter
- Bridge
- Composite
- Decorator
- Facade
- Flyweight
- Proxy

### Behavioral

- Chain of Responsibility
- Command
- Interpreter
- Iterator
- Mediator
- Memento
- [Observer](observer)
- State
- [Strategy](strategy)
- Template
- Visitor
